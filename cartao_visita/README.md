## 🚀 O Projeto
🚀 O Projeto

Este projeto representa o meu primeiro passo no desenvolvimento mobile utilizando o framework Flutter. O objetivo principal desta atividade foi criar um cartão de visita digital personalizado, permitindo a aplicação prática dos conceitos fundamentais de interface de usuário (UI) e a manipulação da árvore de widgets.
Durante o desenvolvimento, foquei no uso de widgets essenciais para a construção de layouts modernos e organizados, tais como:
Column e Row: Utilizados para o alinhamento preciso dos elementos de texto e ícones, garantindo uma estrutura visual limpa.
Container: Aplicado como a base principal do cartão, onde explorei propriedades de decoração, bordas arredondadas e efeitos de profundidade com sombras.
Text e TextStyle: Empregados para a estilização das informações profissionais, ajustando pesos de fonte, tamanhos e cores para uma melhor hierarquia visual.
Como diferencial, o layout foi totalmente personalizado com as cores vibrantes da bandeira do Brasil, criando uma identidade visual marcante e patriótica. Além da parte estética, o projeto conta com interatividade total nos links externos, integrando meu portfólio de forma funcional.
=========================================================================================
🌟 Diferenciais do Meu Projeto

Design Nacional: Interface vibrante inspirada no Brasil com acabamento em BoxShadow e bordas arredondadas.
Experiência Real: Botões clicáveis que utilizam o pacote url_launcher para abrir perfis externos.
Arquitetura: Código organizado em componentes (StatelessWidgets) para facilitar a manutenção.
